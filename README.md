# Clinic Frontend PrimeNG

Este es el frontend del sistema de gestión clínica desarrollado con **Angular 19** y **PrimeNG**. Contiene las vistas de **Login** y **Register**.

## 🚀 Tecnologías utilizadas

- Angular 19
- PrimeNG
- PrimeIcons
